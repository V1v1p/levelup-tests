##### 우분투 패키지 설치 방법#####

**1) 패키지 목록 최신 상태로 갱신하기**

```bash
sudo apt update
```

```tech
apt
```

**2) 웹서버(apache2) 설치하기**

```bash
sudo apt install apache2
```

```tech
apt
```

**3) DB서버(mariadb) 설치하기**

```bash
sudo apt install mariadb-server
```

```tech
apt
```

**4) 설치된 패키지 확인**

```bash
apt list --installed | grep apache2
```

```tech
apt
```

**5) 설치된 패키지 확인**

```bash
dpkg -l | grep mariadb-server
```

```tech
dpkg
```

**6) 설치된 패키지(서비스) 정상 동작 확인**

```bash
sudo systemctl status apache2
```

```tech
systemctl
```
