##### 쉘 프롬프트 형식/색 어떻게 바꾸나요 ? #####

**1) PS1 환경변수 수정 (예: [유저명@호스트명:현재경로]$)**

```bash
echo 'export PS1="\[\u@\h:\w\]\$ "' >> ~/.bashrc
```

```tech
prompt,bash
```

**2) 프롬프트에 색상 추가**

```bash
echo 'export PS1="\[$(tput setaf 2)\][\u@\h:\w]\$ \[$(tput sgr0)\]"' >> ~/.bashrc
```

```tech
prompt,bash
```

**3) 변경사항 적용**

```bash
source ~/.bashrc
```

```tech
source,prompt
```
