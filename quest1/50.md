##### 파일 찾기 locate #####

**1) locate 설치**

```bash
sudo dnf install -y mlocate
```

```tech
locate,dnf
```

**2) 실습용 파일 생성**
```bash
touch myfile.txt
```

**3) 파일 DB 업데이트**

```bash
sudo updatedb
```

```tech
locate
```

**4) 파일명으로 검색**

```bash
locate myfile.txt
```

```tech
locate
```

**5) 검색 결과 개수 제한**

```bash
locate -n 10 "*.log"
```

```tech
locate
```

**6) 실습용 파일 삭제**
```bash
rm myfile.txt
```
