##### 자주 쓰는 명령어 별칭(alias) 설정하기 #####

**1) .bashrc 파일 열기**

```bash
vi ~/.bashrc
```

```tech
vi,alias
```

**2) 별칭 추가 (예: ll='ls -l')**

```bash
alias ll='ls -l'
```

```tech
alias
```

**3) 변경사항 적용**

```bash
source ~/.bashrc
```

```tech
source,alias
```

**4) 설정된 별칭 확인**

```bash
alias
```

```tech
alias
```
