##### 부팅 로그는 어떻게 보나요?#####

**1) 현재 시스템의 전체 부팅 로그 보기(q로 종료)**

```bash
journalctl -b
```

```tech
systemd,journalctl
```

**2) 이전 부팅 시의 로그 보기**

```bash
journalctl -b -1
```

```tech
systemd,journalctl
```

**3) 커널 관련 부팅 로그만 보기**

```bash
journalctl -k
```

```tech
systemd,journalctl
```

**4) 특정 시간 이후의 부팅 로그 보기 (예: 오늘 9시 이후)**

```bash
journalctl --since "09:00"
```

```tech
systemd,journalctl
```

**5) 부팅 로그를 실시간으로 모니터링**

```bash
journalctl -f
```

```tech
systemd,journalctl
```

**6) 부팅 로그를 페이지 단위로 보기(enter로 다음 페이지)**

```bash
journalctl -b | less
```

```tech
systemd,journalctl
